Additional Details

This section of the documentation provides further information on contributing, frequently asked questions, the changelog, acknowledgments, and how to get support.

Contributing

  • Clone the CoRaLS repository
  • Create a new branch (or fork for larger changes)
  • Locally make your changes
  • Commit changes and push to your repository
  • Submit a pull request with a clear description of the changes.

Unit Testing

Unit testing is currently in development. In the test directory, you can find the current test files. The basicunittests.jl file contains unit testing for constants.jl, geometry.jl, fresnel.jl and ice.jl. Testing for the remaining files is in progress. When files are completed they can be added to the runtests.jl file.

You can run all completed tests by executing include("test/runtests.jl");
